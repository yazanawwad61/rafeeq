# Rafeeq — Student Housing Platform

A map-based roommate matching platform built specifically for Amman, Jordan.

## The Problem
Students in Jordan find roommates through chaotic Facebook groups with no filtering, no trust system, and no structure.

## The Solution
Rafeeq is a verified, map-based platform where users can browse available apartments across Amman, filter by location, gender preference, price, and lifestyle tags, and contact owners directly through the platform.

## Features
- Interactive Google Maps with listing pins
- Gender preference filtering
- Lifestyle tags (Doesn't Smoke, Prays, Quiet Lifestyle, etc.)
- Admin approval system for listing verification
- National ID verification for listing owners
- Built-in messaging system (coming)

## Tech Stack
- Backend: Python 3.11 + Flask
- Database: SQLite
- Frontend: HTML + CSS + JavaScript
- Maps: Google Maps JavaScript API
